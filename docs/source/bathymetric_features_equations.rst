Bathymetry Features Equations
-----------------------------


.. math::
   :label: tpi-threshold-high

    TPI\_threshold = mean\_TPI + c * STD\_TPI


.. math::
   :label: tpi-threshold-low

    TPI\_threshold = mean\_TPI - c * STD\_TPI


.. math::
   :label: lmi-threshold

    LMI\_threshold = mean\_LMI + c * STD\_LMI


.. math::
   :label: no-threshold

   NO\_threshold = mean\_NO - c * STD\_NO


.. math::
   :label: po-threshold

   PO\_threshold = mean\_PO - c * STD\_PO


.. math::
   :label: ci-threshold

   CI\_threshold = mean\_CI - c * STD\_CI
